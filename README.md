To run the project please run the following commands:
1. open vscode
2. open new terminal in vscode
3. run the following commands:
    3.1. git clone https://github.com/PrathamAsrani/DashBoard.git
    3.2. cd DashBoard
    3.3. npm install
    3.4. npm start
